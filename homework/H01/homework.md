### 2.4
生成一个5阶的Hilbert矩阵,
$H=(h_{ij})_{n \times n}$, $h_{ij}=\frac{1}{i+j-1}$,$i,j = 1,2, \ldots, n.$

```r
n <- 5
H <- matrix(0, n, n)
for (i in 1:n) {
    for (j in 1:n) {
        H[i, j] <- 1/(i + j - 1)
    }
}
H
```
